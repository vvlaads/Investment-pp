# Инструкция по запуску проекта

## Требования

Перед запуском убедитесь, что установлены:

- **Node.js** (LTS версия)
- **npm** (устанавливается вместе с Node.js)
- **Expo Go** (для запуска на телефоне)

Проверка:
```bash
node -v
npm -v
```

## Установка зависимостей
1. Открыть терминал
2. Перейти в папку проекта:
   ```bash
   cd 2-cross-platorm-app
   ```
3. Установить зависимости:
    ```bash
    npm install
    ```

## Запуск
```bash
npm start
```

## Запуск на телефоне
1. Установить Expo Go
2. Отсканировать QR-код из терминала
3. Дождаться запуска приложения
